# Resultados.zip
Ao descompactar o arquivo Resultados.zip, serão criadas as seguintes subpastas M1 e M2.

Dentro de cada uma das subpastas M1 e M2 existem outras duas subpastas Classificacao e Inferencia.

Dentro da subpasta Classificacao tem todos resultados das classificações e dentro da subpasta Inferencia tem todos resultados das inferencias.
